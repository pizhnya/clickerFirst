# Iso Clicker Runner (3D prototype)

Улучшенный прототип с более "игровой" прогрессией:

- 10 стейджей на зону, 10-й стейдж — босс.
- Автобег -> контакт с врагом -> клик-бой -> переход к следующему стейджу.
- Сохранение прогресса в `localStorage`.
- Две ветки улучшений: клик-урон и idle DPS.
- Критические клики (шанс/множитель).

## Формулы

- `enemyHp(zone, stage) = ceil((10 * (zone + 1.55^zone)) * (1 + stage*0.12) * bossMultiplier)`
- `bossMultiplier = 8`, если `stage = 10`, иначе `1`
- `enemyGold(zone, stage) = ceil((4 * 1.15^(zone-1)) * (1 + stage*0.08))`

## Что и где менять под свои модели

- Игрок: блок `hero` в `script.js`.
- Враг: `spawnEnemy()` в `script.js`.

Можно заменить процедурные меши на загрузку своих GLB/FBX без изменения общей логики.


## Окружение

Добавлено простое процедурное окружение вокруг дороги: серые камни разного размера и упрощённые деревья (ствол + крона) со случайным масштабом и позицией по обе стороны трека.
